## Projects

### [World Astronomy](https://github.com/Tanweer-Ahmed-Chiktay/Astronomy)
Real-time 3D solar system built with Three.js and Next.js. Planet positions computed from NASA J2000 orbital elements, ISS tracked live every 8 seconds, and a seamless scroll-to-zoom into Google Photorealistic 3D Tiles — no button press, just proximity-driven crossfade from interplanetary scale to street level.

`Three.js` `Next.js` `TypeScript` `NASA orbital mechanics` `Google 3D Tiles` `WebGL`

### [LyrnOS](https://github.com/Tanweer-Ahmed-Chiktay/LyrnOS)
Adaptive learning studio for Grade 10 learners. Live AI tutor with Excalidraw whiteboard (PNG export for vision analysis), bidirectional voice sessions via AWS Nova Sonic, PhET simulation lab with browser-agent integration, and reasoning-first work analysis powered by Claude Haiku through Amazon Bedrock.

`AWS Bedrock` `Nova Sonic` `Clerk` `Supabase` `Vite` `TypeScript`

### [Agently](https://github.com/Tanweer-Ahmed-Chiktay/agently-public) · [Live →](https://agently-ten.vercel.app)
Legal AI platform for law firms. AI chat with streaming, browser automation for court e-filing (Steel CDP + Playwright), document management on S3, and a custom Aurora DSQL connection pool with IAM token rotation. Built for the H0 Hackathon on Vercel + AWS.

`Aurora DSQL` `AWS Bedrock` `Express 5` `React 19` `Clerk` `Playwright` `Steel.dev`

### [Khanya AI](https://github.com/Tanweer-Ahmed-Chiktay/Khanya_ai)
AI tutor built for the South African CAPS curriculum — essay writer, AI whiteboard, and a code playground. Designed for learners without access to expensive tutoring.
